# Genshin Impact Data Project

This project contains JSON data files and accompanying processing tools related to the Genshin Impact game.

## Project Overview

This repository is dedicated to collecting and organizing various data from the Genshin Impact game, including but not limited to:

- Item data (Item.json)
- Teleport point data (teleport_tasks.json)
- Character data processing tool (avatarlist.cpp)

## Data Details

### Item Data (Item.json)

A collection of data containing information on various in-game items, weapons, artifacts, and more.

### Teleport Task Data (teleport_tasks.json)

Contains data on in-game teleport points and associated tasks.

### Character List Processing (avatarlist.cpp)

A C++ program designed to process and parse character data.
